## 學到的教訓

### OpenClaw 模型配置
1. **兩層結構：**
   - `models.providers` - 模型定義（告訴系統有哪些模型）
   - `agents.defaults.models` - 模型啟用（告訴系統哪些可以被選擇）

2. **別名配置：**
   - 有別名：`"zai/glm-5": { "alias": "GLM" }` → 可用 GLM
   - 無別名：`"zai/glm-4.7": {}` → 只能用完整 ID

3. **配置順序：**
   - 先定義提供商和模型
   - 再添加 auth 配置
   - 最後重啟生效
