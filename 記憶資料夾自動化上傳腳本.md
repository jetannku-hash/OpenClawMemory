# 記憶資料夾自動化上傳腳本

## 📁 來源與目標

### 來源目錄
**記憶資料夾：** `/home/terryku/.openclaw/workspace/memory`
```
memory/
├── 2026-02-26.md
├── 2026-02-28.md
├── 2026-03-01.md
├── 2026-03-02.md
├── 2026-03-02_gog-calendar-rules.md
└── 2026-03-03.md
```

### 目標倉庫
**OpenClawMemory：** https://github.com/jetannku-hash/OpenClawMemory

---

## 🚀 自動化功能

### 1. 批量上傳記憶檔案
自動將 `memory/` 目錄下的所有 `.md` 檔案上傳到 OpenClawMemory 倉庫

### 2. Base64 編碼
每個檔案都會先進行 Base64 編碼，確保正確處理 UTF-8 內容

### 3. 進度顯示
上傳過程中會顯示：
- 📝 正在上傳：`[檔名]`
- ✅ 上傳成功：`[檔名]`
- ❌ 上傳失敗：`[檔名] - [錯誤訊息]`
- 📊 統計：`[已上傳 / 總數]`

---

## 🛠️ 使用說明

### 腳本執行
```bash
bash /tmp/batch_upload_memory.sh
```

### 自動化功能
- ✅ 自動檢測記憶檔案
- ✅ 自動 Base64 編碼
- ✅ 自動構建 API 請求
- ✅ 自動處理 API 錯誤
- ✅ 自動顯示上傳進度
- ✅ 自動生成上傳報告
